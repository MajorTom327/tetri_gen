 TETRI GEN
Il s'agit d'un generateur de piece de tetriminos utilisable dans le cadre du projet #FillIt de 42
Celui ci genere un nombre de tetriminos defini en parametre.

Utilisation:
$> make
$> ./tetri_gen 21 > fichier
